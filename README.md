CASIA – Neuro-Symbolic AI Guard System

CASIA (Casual Awareness self improving  Assistant) is a hybrid AI system designed to improve the reliability and safety of Large Language Model (LLM) outputs. The system integrates neural language models with symbolic reasoning modules to detect hallucinations, verify factual information, and provide explainable responses.

CASIA introduces a layered validation architecture where generated responses are evaluated through rule-based reasoning, knowledge verification, and external information retrieval before being returned to the user.

Overview

CASIA is a web-based AI system consisting of:

Frontend
A Streamlit web interface that allows users to interact with the AI system, submit queries, and upload documents.

Backend
A FastAPI service responsible for handling requests, running AI reasoning modules, and generating responses.

The system runs locally on a web server and can be accessed through a browser.

Key Features

Hallucination Detection
Detects potentially unreliable or fabricated responses generated by LLMs.

Neuro-Symbolic Reasoning
Combines neural network outputs with symbolic reasoning rules for validation.

Knowledge Graph Integration
Verifies factual consistency using structured knowledge sources.

Web Search Verification
Retrieves external information to validate responses.

Explainable AI Tracing
Provides reasoning traces explaining how the system produced an answer.

Confidence Scoring
Assigns reliability scores to responses based on validation layers.

Document Query System
Allows users to upload documents (PDF, DOCX, TXT) and query them using AI.
